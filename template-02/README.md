# template-02

Template Astro con Tailwind CSS, derivato dal design "Rose Burger" (shuffle).

## Sviluppo

```bash
npm install
npm run dev
```

Il server parte su http://localhost:4322.

## Build

```bash
npm run build
npm run preview
```

## Struttura

Stessa architettura di `template-01`: contenuti JSON da `clients/{CLIENT_SLUG}/`, schema Zod, Decap CMS integrato, multilingua IT/EN.

La differenza è il design: questo template usa **Tailwind CSS** con un look dark/moderno.
